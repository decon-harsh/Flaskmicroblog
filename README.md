# Requirements:
1) Flask
2) SqlAlchemy
3) WtForms
4) Pillow     

# Installations:
1) Clone this repo
2) Now move into this directory
3) $python3 run.py

# To not run into debug  mode:
1) open __init__.py
2) run.debug=FALSE
